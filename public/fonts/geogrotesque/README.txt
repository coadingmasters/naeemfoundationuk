Geogrotesque — self-hosted web font
====================================

Geogrotesque is a licensed commercial font (Emtype Foundry). It cannot be
downloaded automatically, so its files must be placed here manually.

Put your licensed WEB font files (woff2 preferred, woff as fallback) in this
folder using EXACTLY these filenames:

    geogrotesque-400.woff2   geogrotesque-400.woff     (Regular)
    geogrotesque-500.woff2   geogrotesque-500.woff     (Medium)
    geogrotesque-600.woff2   geogrotesque-600.woff     (SemiBold)
    geogrotesque-700.woff2   geogrotesque-700.woff     (Bold)
    geogrotesque-800.woff2   geogrotesque-800.woff     (Bold / Black — used by headings)

Notes
-----
- Only .woff2 is strictly required; .woff is an optional older-browser fallback.
- If you only have a couple of weights, add just those — the others will fall
  back gracefully to the nearest available weight.
- If you have an Adobe Fonts (Typekit) kit for Geogrotesque instead of files,
  tell the developer and it can be wired via the kit <link> instead.

Until real files are added here, the site automatically falls back to
Instrument Sans, so text always looks correct.
